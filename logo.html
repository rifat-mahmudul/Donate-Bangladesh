Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model represents an HTML document as a tree, where each node is an object corresponding to a part of the document (such as an element, attribute, or text).

How do you select an element from the DOM?

We can select an element from the DOM (Document Object Model) in JavaScript in many ways. Here are the most commonly used methods:
1. document.getElementById();
2. document.getElementsByClassName();
3. document.getElementsByTagName();
4. document.querySelector();
5. document.querySelectorAll();

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is one of the most popular technique in JavaScript where instead of attaching event listeners to individual child elements, a single event listener is attached to a common parent element. The event is then "delegated" to the child elements using event bubbling (or event propagation). When an event occurs on a child element, it bubbles up through the DOM and can be captured by the parent, allowing you to determine which specific child element triggered the event.
